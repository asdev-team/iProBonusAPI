### iProBonus API

#### Установка
###### `git clone https://github.com/asdev-team/iProBonusAPI.git`
###### `npm i`

#### Скрипты

##### Запуск приложения в development mode.
###### `npm run start`
Открыть [http://lvh.me:8080/](http://lvh.me:8080/) в браузере.

##### Сборка приложения
###### `npm run build`
